At its core, GitHub is a cloud platform built around Git, which is a distributed version control system.

Think of it like this:

Git = engine (tracks changes in your code)

GitHub = platform (hosts that code online + adds collaboration features)

So instead of keeping your project only on your laptop, GitHub lets you:

Store it online (repositories / “repos”)

Track every change (commits, history)

Collaborate with others easily
